# 接口文档

## 一、企业管理模块

### 1. 获取企业列表

- **接口：** `getEnterpriseInfoApi`
- **请求方式：** GET
- **请求参数：**

| 参数名   | 类型   | 必填 | 说明         |
| -------- | ------ | ---- | ------------ |
| name     | string | 否   | 企业名称     |
| page     | number | 是   | 当前页码     |
| pageSize | number | 是   | 每页条数     |

- **返回数据：**

| 字段名         | 类型   | 说明       |
| -------------- | ------ | ---------- |
| data.rows      | array  | 企业列表   |
| data.total     | number | 总条数     |

---

### 2. 删除企业

- **接口：** `deleteEnterpriseAPI`
- **请求方式：** DELETE
- **请求参数：** 企业 ID

---

### 3. 获取租赁楼宇列表

- **接口：** `getRentBuildListAPI`
- **请求方式：** GET
- **返回数据：** 楼宇列表（id、name）

---

### 4. 获取合同列表

- **接口：** `getRentListAPI`
- **请求方式：** GET
- **请求参数：** 企业 ID
- **返回数据：** 合同列表（租赁楼宇、起止时间、合同状态等）

---

### 5. 上传合同文件

- **接口：** `uploadAPI`
- **请求方式：** POST
- **请求参数：** FormData（合同文件，支持 .doc .docx .pdf，不超过 5M）

---

### 6. 退租

- **接口：** `outRentAPI`
- **请求方式：** PUT
- **请求参数：** 合同 ID（contractId）

---

## 二、停车计费规则模块

### 1. 获取计费规则列表

- **接口：** `calculateCardApi`
- **请求方式：** GET
- **请求参数：**

| 参数名   | 类型   | 必填 | 说明     |
| -------- | ------ | ---- | -------- |
| page     | number | 是   | 当前页码 |
| pageSize | number | 是   | 每页条数 |

- **返回数据：**

| 字段名         | 类型   | 说明           |
| -------------- | ------ | -------------- |
| data.rows      | array  | 计费规则列表   |
| data.total     | number | 总条数         |

- **rows 数据结构：**

| 字段名        | 类型   | 说明                                    |
| ------------- | ------ | --------------------------------------- |
| id            | number | 规则编号                                |
| ruleNumber    | string | 计费规则编号                            |
| ruleName      | string | 计费规则名称                            |
| freeDuration  | number | 免费时长(分钟)                          |
| chargeCeiling | number | 收费上限(元)                            |
| chargeType    | string | 计费方式（duration/turn/partition）     |
| ruleNameView  | string | 计费规则描述                            |

- **chargeType 枚举值：**

| 值        | 说明     |
| --------- | -------- |
| duration  | 时长收费 |
| turn      | 按次收费 |
| partition | 分段消费 |

---

> **说明：** 以上接口文档基于项目现有代码整理，如后续新增接口请及时补充更新。
