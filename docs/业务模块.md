# 业务模块说明

## 一、企业管理模块

### 1. 模块概述

**路径：** `src/views/park/ParkEnterprise.vue`  
**功能：** 管理园区入驻企业信息，包括企业的增删改查、合同管理等。

### 2. 页面结构

```
┌──────────────────────────────────────────┐
│ 搜索区域（企业名称 + 查询按钮）            │
├──────────────────────────────────────────┤
│ 操作区域（添加企业按钮）                   │
├──────────────────────────────────────────┤
│ 表格区域（企业列表 + 可展开合同子表格）     │
│  ├─ 序号                                  │
│  ├─ 企业名称                              │
│  ├─ 联系人                                │
│  ├─ 联系电话                              │
│  ├─ 操作（添加合同 / 查看 / 编辑 / 删除）  │
│  └─ 展开行：合同子表格                     │
│      ├─ 租赁楼宇                           │
│      ├─ 租赁起始时间                       │
│      ├─ 合同状态（Tag 标签）               │
│      └─ 操作（续租 / 退租 / 删除）         │
├──────────────────────────────────────────┤
│ 分页区域                                   │
├──────────────────────────────────────────┤
│ 弹窗：添加合同表单                         │
│  ├─ 租赁楼宇（下拉选择）                   │
│  ├─ 租赁起止日期（日期范围选择器）          │
│  └─ 租赁合同（文件上传）                   │
└──────────────────────────────────────────┘
```

### 3. 功能详情

#### 3.1 企业列表展示

- 使用 `el-table` 分页展示企业信息
- 请求参数：`{ name, page, pageSize }`
- 接口：`getEnterpriseInfoApi`

#### 3.2 企业搜索

- 根据企业名称进行模糊查询
- 搜索时重置页码为 1
- 清空搜索条件后自动刷新列表

#### 3.3 企业增删改查

- **添加：** 跳转路由 `/exterpriseAdd`
- **编辑：** 跳转路由 `/exterpriseAdd?id=xxx`（复用新增页面）
- **查看：** 跳转路由 `/exterpriseDetail?id=xxx`
- **删除：** 使用 `ElMessageBox.confirm` 二次确认后调用 `deleteEnterpriseAPI`

#### 3.4 合同管理（展开行）

- 使用 `el-table` 的 `type="expand"` 展开行展示合同子表格
- 通过 `row-key="id"` + `:expand-row-keys="expandRowKeys"` 控制展开状态
- 点击展开时调用 `getRentListAPI` 获取该企业的合同列表

#### 3.5 合同状态映射

| 状态值 | 中文展示 | Tag 类型 |
| ------ | -------- | -------- |
| 0      | 待生效   | info     |
| 1      | 生效中   | success  |
| 2      | 已到期   | warning  |
| 3      | 已退租   | danger   |

#### 3.6 添加合同（弹窗表单）

- 使用 `el-dialog` + `el-form` 实现弹窗表单
- 表单字段：
  - **租赁楼宇：** 下拉选择，数据来源 `getRentBuildListAPI`
  - **租赁起止日期：** `el-date-picker` 日期范围选择器，格式 `YYYY-MM-DD`
  - **租赁合同：** `el-upload` 手动上传模式，支持 `.doc .docx .pdf`，不超过 5M
- 合同文件通过 `uploadAPI` 上传后获取 `contractId`

#### 3.7 退租

- 调用 `outRentAPI`，传入 `contractId`
- 退租后合同状态变为"已退租"，退租和删除按钮禁用

### 4. 涉及接口

| 接口                   | 方式   | 说明             |
| ---------------------- | ------ | ---------------- |
| `getEnterpriseInfoApi` | GET    | 获取企业列表     |
| `deleteEnterpriseAPI`  | DELETE | 删除企业         |
| `getRentBuildListAPI`  | GET    | 获取租赁楼宇列表 |
| `getRentListAPI`       | GET    | 获取合同列表     |
| `uploadAPI`            | POST   | 上传合同文件     |
| `outRentAPI`           | PUT    | 退租操作         |

## 二、停车计费规则管理模块

### 1. 模块概述

**路径：** `src/views/car/CarRule.vue`  
**功能：** 管理停车场计费规则，包括规则的展示、导出等。

### 2. 页面结构

```
┌────────────────────────────────────────────────┐
│ 操作区域（增加停车计费规则 + 导出Excel）          │
├────────────────────────────────────────────────┤
│ 表格区域（计费规则列表）                         │
│  ├─ 序号                                        │
│  ├─ 计费规则编号（ruleNumber）                   │
│  ├─ 计费规则名称（ruleName）                     │
│  ├─ 免费时长(分钟)（freeDuration）               │
│  ├─ 收费上限(元)（chargeCeiling）                │
│  ├─ 计费方式（chargeType → 枚举映射）            │
│  ├─ 计费规则（ruleNameView）                     │
│  └─ 操作（编辑 / 删除）                          │
├────────────────────────────────────────────────┤
│ 分页区域                                         │
└────────────────────────────────────────────────┘
```

### 3. 功能详情

#### 3.1 计费规则列表展示

- 使用 `el-table` 分页展示计费规则
- 请求参数：`{ page, pageSize }`
- 接口：`calculateCardApi`

#### 3.2 计费方式枚举映射

| 枚举值    | 中文展示 |
| --------- | -------- |
| duration  | 时长收费 |
| turn      | 按次收费 |
| partition | 分段消费 |

#### 3.3 导出 Excel

- 使用 `xlsx` 库实现前端导出
- 导出流程：

  1. 创建工作簿 `XLSX.utils.book_new()`
  2. 按表头顺序过滤并重组数据
  3. 将枚举值 `chargeType` 转换为中文文本
  4. 转换为工作表 `XLSX.utils.json_to_sheet()`
  5. 自定义中文表头覆盖默认表头
  6. 触发下载 `XLSX.writeFile()`

- 导出字段映射：

| 数据字段      | Excel 表头     |
| ------------- | -------------- |
| id            | 编号           |
| ruleNumber    | 计费规则编号   |
| ruleName      | 计费规则名称   |
| freeDuration  | 免费时长(分钟) |
| chargeCeiling | 收费上限(元)   |
| chargeType    | 计费方式       |
| ruleNameView  | 计费规则       |

#### 3.4 待开发功能

- 🔲 增加计费规则（按钮已预留）
- 🔲 编辑计费规则（按钮已预留）
- 🔲 删除计费规则（按钮已预留）

### 4. 涉及接口

| 接口               | 方式 | 说明             |
| ------------------ | ---- | ---------------- |
| `calculateCardApi` | GET  | 获取计费规则列表 |

### 5. 涉及类型

```typescript

  freeDuration: number
```
